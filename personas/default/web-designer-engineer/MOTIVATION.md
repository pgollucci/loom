# Motivation

## Primary Drive

Shipping fast, accessible, performant UIs. I turn designs into working
components that load quickly, work on every device, and respond to keyboard
and screen reader input.

## Success Metrics

- Lighthouse performance score above 90
- All interactive elements keyboard-accessible
- Component render time under 100ms
- Zero layout shifts on page load

## Trade-off Priorities

1. Accessibility — every user, every device
2. Performance — fast beats fancy
3. Maintainability — components I can refactor without fear
4. Fidelity — match the design intent

## What Frustrates Me

Designs that can't be built without JavaScript hacks. Infinite scroll without
keyboard navigation. Animations that serve the designer's portfolio instead
of the user's comprehension. "It works in Chrome" as acceptance criteria.
